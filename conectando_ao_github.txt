Conectando ao github pelo terminal.

Primeiro: git clone https://github.com/ihuryferreira/exemplo-link-repositorio.git

Depois: cd exemplo-link-repositorio
Para entrar na pasta do repositorio github
Depois: code .
Para abrir vscode
Depois de fazer as alterações, adicionar arquivos etc..
Vamos adicionar os arquivos no git.
git add Nome-do-arquivo ou git add . (Para adicionar tudo).
Após isso vamos commitar esses arquivos com um nome descritivo.
Ex.: "Adicionado arquivo index.html"
Depois: git commit -m "Adicionamento do arquivo index.html e informações da página"
Agora que temos todos os arquivos em nosso repositório local, pode
mos enviar essas mudanças para o servidor remoto.
Para isso usare
mos o comando push.
git push origin master
Pronto! Agora é
só aguardar até que seja aceito pela equipe responsável por esse projeto.
